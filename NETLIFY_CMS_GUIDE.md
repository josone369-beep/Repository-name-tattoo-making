     1	# 🎯 Netlify CMS 설정 가이드
     2	
     3	타투메이킹 홈페이지에 관리자 페이지를 추가하는 방법입니다.
     4	
     5	---
     6	
     7	## ✅ 완료된 작업
     8	
     9	- ✅ `admin/index.html` - 관리자 페이지 생성됨
    10	- ✅ `admin/config.yml` - CMS 설정 완료
    11	- ✅ `index.html` - Netlify Identity 위젯 추가됨
    12	
    13	---
    14	
    15	## 🚀 설정 단계
    16	
    17	### **Step 1: GitHub 계정 만들기**
    18	
    19	1. https://github.com/ 접속
    20	2. "Sign up" 클릭
    21	3. 이메일, 비밀번호 입력
    22	4. 계정 생성 완료
    23	
    24	---
    25	
    26	### **Step 2: GitHub에 프로젝트 업로드**
    27	
    28	#### A. GitHub 웹사이트에서 하기 (쉬움)
    29	
    30	1. GitHub 로그인
    31	2. 우측 상단 "+" → "New repository" 클릭
    32	3. Repository name: `tattoo-making`
    33	4. Public 선택
    34	5. "Create repository" 클릭
    35	
    36	#### B. 파일 업로드
    37	
    38	1. "uploading an existing file" 클릭
    39	2. 모든 프로젝트 파일 드래그 앤 드롭
    40	   ```
    41	   - index.html
    42	   - admin/ (폴더)
    43	   - css/ (폴더)
    44	   - js/ (폴더)
    45	   - README.md
    46	   ```
    47	3. "Commit changes" 클릭
    48	
    49	---
    50	
    51	### **Step 3: Netlify 연결**
    52	
    53	1. https://app.netlify.com/ 접속
    54	2. "Add new site" → "Import an existing project"
    55	3. "GitHub" 선택
    56	4. `tattoo-making` 저장소 선택
    57	5. "Deploy site" 클릭
    58	
    59	**잠시 기다리면 배포 완료!**
    60	
    61	---
    62	
    63	### **Step 4: Netlify Identity 활성화**
    64	
    65	1. Netlify 대시보드에서 사이트 클릭
    66	2. 좌측 메뉴 "Site configuration" → "Identity" 클릭
    67	3. "Enable Identity" 버튼 클릭
    68	
    69	---
    70	
    71	### **Step 5: Git Gateway 활성화**
    72	
    73	1. Identity 페이지에서 "Services" 탭
    74	2. "Git Gateway" → "Enable Git Gateway" 클릭
    75	
    76	---
    77	
    78	### **Step 6: 초대 전용 모드 설정**
    79	
    80	1. Identity 페이지 → "Settings and usage"
    81	2. "Registration" → "Invite only" 선택
    82	   (아무나 가입 못하게!)
    83	
    84	---
    85	
    86	### **Step 7: 관리자 계정 만들기**
    87	
    88	1. Identity 페이지 → "Invite users"
    89	2. 본인 이메일 입력
    90	3. "Send" 클릭
    91	4. 이메일 확인 → "Accept the invite" 클릭
    92	5. 비밀번호 설정
    93	
    94	---
    95	
    96	## 🎉 완료! 이제 사용 가능
    97

   105	
   106	- 📸 **포트폴리오** - 작업물 추가/수정/삭제
   107	- 🖼️ **메인 슬라이더** - Hero 이미지 관리
   108	- 🎨 **작업 도안** - 도안 이미지 관리
   109	- ⭐ **고객 후기** - 리뷰 관리
   110	- ⚙️ **홈페이지 설정** - 연락처, 가격 수정

   114	## 💡 사용 방법
   115	
   116	### **1. 포트폴리오 이미지 추가**
   117	
   118	1. 관리자 페이지 로그인
   119	2. "포트폴리오" 클릭
   120	3. "New 포트폴리오" 클릭
   121	4. 정보 입력:
   122	   - 제목: "이레즈미 - 용"
   123	   - 카테고리: irezumi 선택
   124	   - 설명: "풀슬리브"
   125	   - 이미지: 클릭해서 업로드
   126	5. "Publish" 클릭
   127	6. 자동으로 사이트에 반영!
   128	
   129	### **2. 메인 슬라이더 이미지 변경**
   130	
   131	1. "메인 슬라이더" 클릭
   132	2. "New 메인 슬라이더" 클릭
   133	3. 정보 입력:
   134	   - 캡션: "블랙&그레이 리얼리즘"
   135	   - 이미지: 업로드
   136	   - 순서: 1
   137	4. "Publish" 클릭
   138	
   139	### **3. 연락처 수정**
   140	
   141	1. "홈페이지 설정" 클릭
   142	2. "연락처 정보" 클릭
   143	3. 전화번호, 주소 등 수정
   144	4. "Publish" 클릭
   145	
   146	### **4. 가격 수정**
   147	
   148	1. "홈페이지 설정" 클릭
   149	2. "가격 정보" 클릭
   150	3. 가격 수정
   151	4. "Publish" 클릭

   183	## 🆘 문제 해결
   184	
   185	### **문제: /admin 접속이 안 돼요**
   186	
   187	**해결:**
   188	- admin 폴더가 제대로 업로드됐는지 확인
   189	- Netlify에서 재배포 (Deploys → Trigger deploy)
   190	
   191	### **문제: 로그인이 안 돼요**
   192	
   193	**해결:**
   194	- Netlify Identity가 활성화됐는지 확인
   195	- Git Gateway가 활성화됐는지 확인
   196	- 이메일 초대를 수락했는지 확인
   197	
   198	### **문제: 이미지가 업로드 안 돼요**
   199	
   200	**해결:**
   201	- images/uploads 폴더가 생성되는지 확인
   202	- 이미지 크기를 줄여보기 (5MB 이하)
